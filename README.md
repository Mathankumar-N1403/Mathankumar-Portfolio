# Mathankumar N — Portfolio

A professional React portfolio website built with JSX and CSS.

## 🚀 Tech Stack
- React 18
- react-icons
- Google Fonts (Fraunces + Outfit)

## ⚙️ Setup & Run

npm install  
npm start  
npm run build  

Open http://localhost:3000 in your browser.
